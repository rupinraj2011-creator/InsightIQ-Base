👋 WELCOME! Follow these 3 simple steps:

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: START THE SERVERS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Option A - Automatic (RECOMMENDED)
  ─────────────────────────────────
  Linux/Mac:
    $ ./run.sh

  Windows:
    > run.bat


  Option B - Manual
  ─────────────────
  Terminal 1 (Backend):
    $ cd backend
    $ pip install -r requirements.txt
    $ python app.py

  Terminal 2 (Frontend):
    $ cd frontend
    $ python -m http.server 8080


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 2: OPEN YOUR BROWSER
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    http://localhost:8080/START_HERE.html

STEP 3: CONFIGURE API KEYS (for real data)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Create file: backend/.env

  Add your API keys:
    GNEWS_API_KEY=your_key
    SERPAPI_KEY=your_key
    ALPHAVANTAGE_KEY=your_key
    OPENAI_API_KEY=your_key
    TWITTER_BEARER_TOKEN=your_key
    REDDIT_CLIENT_ID=your_key
    REDDIT_CLIENT_SECRET=your_key
    REDDIT_USER_AGENT=InSightIQ/1.0

  Where to get keys:
    ✓ GNews: https://gnews.io/ (100 free/day)
    ✓ SerpAPI: https://serpapi.com/ (100 free/month)
    ✓ Alpha Vantage: https://www.alphavantage.co/ (25 free/day)
    ✓ OpenAI: https://platform.openai.com/ (paid)
    ✓ Twitter: https://developer.twitter.com/ (free)
    ✓ Reddit: https://www.reddit.com/prefs/apps (free)

Can't see the UI?
  1. Open: frontend/START_HERE.html
  2. Run diagnostics: frontend/diagnostic.html
  3. Check console: Press F12 in browser
  4. Read: TROUBLESHOOTING.md

Backend errors?
  • Verify .env file exists in backend/
  • Check Python version: python --version (need 3.8+)
  • Reinstall: pip install -r requirements.txt

CORS errors?
  • Ensure backend is running
  • Use http://localhost (not file:///)
  • Check Flask-CORS is installed
